# Password Generator

A simple Python script for validating passwords.

## Description

This script provides a password validation function that checks whether a given password meets the following requirements:

- At least 8 characters long
- Alphanumeric (contains only letters and digits)

## Requirements

- Python 3.x

## Usage

Uncomment the desired function in `password.py` and run it:

```bash
python password.py
```

You will be prompted to enter a password. The script will then tell you whether your password meets the requirements.

## Example

```
Enter your password: myPass1234
Your password is set.
```

```
Enter your password: short
Password length should be greater or equal to eight.
```
